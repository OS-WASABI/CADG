◆ HandleGet()

void cadg_rest::UserController::HandleGet ( http_request  message)
overridevirtual

Returns the requested user(s).

HandleGet will return a user or collection of users determined by the parameters in the url path. If there is not url path, it returns all users. IF there is a user id (a number) next in the path, it returns a user with that id, or no user if there isn't one. If a query for name is in the path, it returns all users with the name as a part of their name.

Parameters
messageThe http request to be parsed.
